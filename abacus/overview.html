Abacus Component Library and Framework

Overview

Abacus is a powerful, modular, and lightweight CSS component library built using OOCSS and BEM, for use across all Paddy Power sites and products. It features a powerful mobile-first grid system and incredibly flexible components to accompany.

Using Abacus

It's important to note that Abacus is not designed to be directly integrated into your project's own CSS/Less/Sass, but instead is designed to exist as a separate CSS file to be used within a <link> in your project's <head>, and referenced from CDN.

This is to improve performance across all channels by caching commonly-styled element and components on the client machine, allowing developers to take more control by applying the own CSS afterwards, with correct usage of the Cascade and Specificity.

If you want to start using Abacus on your project, you can use this to get started:


    <link rel="stylesheet" media="all" src="//c.ppstatic.com/abacus/0.28.0/abacus.min.css">

    <script defer src="//j.ppstatic.com/abacus/0.28.0/abacus.js"></script>
