A Bit About Me

Hey! How’s it going? My name is Stephen Coogan, and I’m a Front-End Web Developer from Dublin, Ireland.

I’ve been mucking about with code for nigh on eight years now, but it’s only been in the past four years that I’ve taken it far more seriously. My expertise is deeply rooted in front-end technologies such as HTML, CSS, and JavaScript — including libraries such as jQuery.

A bit about this site

This site is powered by Octopress, an open-source blogging framework designed for Jekyll, and is hosted on Github Pages. The theme is of my own creation and is released under the MIT License.

The theme itself is built using the powerful inuit.css from Harry Roberts, and SASS. I have tried to keep JavaScript to a bare minimum as I feel it brings very little to a theme such as this.

My philosophy

When it comes to software and code, I believe that people should be as open as the technologies they use.

Especially when it comes to front-end web technologies which - by their very nature and design - are the most open and accessible technologies on the planet to learn, and I feel that I should do whatever I can to help others learn how to use them.

When it comes to design, the user should be put on a pedestal. Design so that they have the best experience possible, and they’ll keep coming back.
